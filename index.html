Welcome to jPregel-aws!

The jPregel-aws project comprises a Java-based implementation of Pregel with a package that enables deployment on Amazon Web Services with a minimum of fuss.

The jPregel part of this project was originally written by Peter Cappello. The current version does not tolerate machine failures. The Java package to run jPregel on AWS, originally written by Varsha Parthasarathy, has been replaced and significantly extended by Charles Munger.
